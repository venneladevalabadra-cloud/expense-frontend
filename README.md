# Smart Expense Tracker Frontend

React frontend for the Smart Expense Tracker full-stack project.

## Live Backend

Render backend API:

```txt
https://smart-expense-backend-sbtd.onrender.com
```

Health check:

```txt
https://smart-expense-backend-sbtd.onrender.com/api/expenses/test
```

## Features

- Add expenses
- View all expenses
- Delete expenses
- Total expense dashboard
- Category pie chart
- Monthly spending chart

## Local Run

```bash
npm install
npm start
```

Open:

```txt
http://localhost:3000
```

## Production Build

```bash
npm run build
```

Deploy the generated `build` folder to Netlify.
